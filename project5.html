Facebook Redo


Project Description

The layout of Facebook's profile page, home page, and default profile picture controls user’s actions and behaviors. In this project, wireframes of the current website were created to analyze facebook webpages. Changes were also suggested. The statement, ‘How might we enhance a positive community on Facebook’ was the focus of this project. A positive community is one that is secure, friendly, and accepting.



Facebook causes users to constantly add more friends resulting in strangers to be easily added.

  • On the profile page, the first object seen is likely the friend suggestions due to the location and colour.
  • The website has multiple locations to add friends: it has a whole page named Find Friends and suggests friends under Friend Requests as well.
  • The functional affordance of adding friends is easy as there are multiple locations to do so.

In order to increase security, the Find Friend tab on Facebook and the friend suggestion section on the user’s profile page should be eliminated. By forcing people to manually search for a friend in the search bar would encourage Facebook users to only have close friends. Limiting the sensory affordance of adding people enhances a positive community as close friends usually care for each other..

Facebook cause people to comment impulsively.

  • It is easy to post comments but very hard to look back on comments previously made. This causes users to feel like the comments they make now will not matter in the future, since they never see the comment again. This creates a side effect of people commenting rudely.

A past comments section should be added to the user’s profile page. The past comment section would contain posts that the user previously commented on. Visibility of the user’s past comments would make users more aware of what they say, discouraging them to say anything rude. Moveover, it should be placed close to the center of the page, so it would be easy to spot. Another adjustment to increase friendliness is adding a larger user’s profile picture beside the section to make a post. Seeing a large profile picture of themselves before the user creates a post it enhances a sense of ownership. The user seeing their profile picture before they post it would stimulate a face-to-face interaction. This magnifies a match between the system and the real world, decreasing the mask of the computer screen that makes it easy to be rude.

Facebook also creates ideals in society.

  • When users create an account there is only an option to select either a female or male account. This makes all other genders feel out of place.
  • Also, the default picture of a male has short hair, while the default picture of a female has longer hair.

In order to enhance acceptance, Facebook should not ask whether the user is a male or female when creating a profile. Furthermore, they should use a minimal-detail gender-neutral profile picture as the default for everyone, so that it does not suggest any specific person and conveys that it could be anyone. This would help people who do not identify as male or female feel accepted and enhance the overall inclusiveness of Facebook.
